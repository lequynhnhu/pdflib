Wrapper classes for JDBC, provides connection pooling, safe wrapping for fields used in insert commands, generic utilities, etc.

Usage: the DBConnectionPool is the key class. It reads the JDBC information for Global and creates the connections.

The two key calls are to get a conection using: 

            Connection myCon = DBConnectionPool.getDBConnection();
and to release the connection with 
            DBConnectionPool.returnCon();

Other than using the pool for allocating and releasing connecitons, all the rest is normal JDBC.

Two Exception classes are defined here:

  • {@link com.pfarrell.utils.database.DuplicateKeyException}
  • {@link com.pfarrell.exceptions.RequiredValueMissingException}
